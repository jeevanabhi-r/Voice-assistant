# AI Voice Study Assistant - Output Demo

This document shows what your Voice Assistant project produces when running.

## How It Works

Your project is built with **Gradio** for the UI and uses **Murf AI** for text-to-speech. Here is the complete flow:

```
USER INPUT (Voice/Text) -> Speech Recognition -> AI Response -> Text-to-Speech (Murf AI) -> AUDIO OUTPUT
```

## Step-by-Step Output Flow

### 1. User Speaks/Types Input
- You can click the "Speak Now" button and record your voice, OR
- Type a question in the text input box

### 2. Speech Recognition Converts Voice to Text
- The audio is processed using SpeechRecognition
- Your spoken words are converted to text (e.g., "explain photosynthesis")

### 3. AI Generates a Personalized Response
- Based on the selected persona (Friendly or Academic), the AI creates a response
- **Friendly Mode**: Simple, encouraging explanations with real-world examples
- **Academic Mode**: Detailed, professional explanations with formal terminology

### 4. Murf AI Converts Response to Speech
- The text response is sent to Murf AI's TTS API
- Voice: "en-US-natalie" (a natural American English voice)
- Format: MP3 audio

### 5. Audio Output is Played
- The user hears the AI response spoken aloud
- An audio player is available to replay the response

## Example Output Scenarios

### Scenario 1: Friendly Mode
- **User Input**: "What is gravity?"
- **AI Text Response**: The assistant responds in a friendly, encouraging tone explaining gravity in simple terms
- **AI Audio**: Spoken by Murf AI's "Natalie" voice as an MP3 file

### Scenario 2: Academic Mode
- **User Input**: "Explain Newton's laws of motion"
- **AI Text Response**: Detailed academic explanation with precise terminology
- **AI Audio**: Professional narration of the academic response

## Gradio UI Output

When you run `python file.py`, a web interface opens at `http://127.0.0.1:7860` with:

- A pulsing animated circle (red gradient) that acts as a visual mic indicator
- Audio recorder for voice input
- Text input box for typing
- Persona selector (Friendly / Academic modes)
- Output area showing the AI text response
- Audio player with the spoken AI response

## How to See the Output Locally

To actually see and hear the output, run these commands on your computer:

```bash
# Install dependencies
pip install -r requirements.txt

# Run the voice assistant
python file.py
```

Then open your browser at: `http://127.0.0.1:7860`

## Terminal Output

When running, you will see in your terminal:

```
Running on local URL: http://127.0.0.1:7860
Murf Status: 200
Murf TTS completed successfully!
```

## API Integrations Used

1. **SpeechRecognition API** - Converts speech to text
2. **Murf AI API** - Converts text to high-quality speech
3. **pyttsx3** - Offline text-to-speech fallback

## Output Files Generated

- `output.mp3` - Generated by pyttsx3 (fallback TTS)
- Audio URL from Murf AI - High-quality TTS output
